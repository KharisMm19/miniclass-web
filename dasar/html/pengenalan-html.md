## 2. Cara Penulisan

Sintaks atau cara penulisan HTML biasanya terlihat seperti ini:

```html
<!DOCTYPE html>
<html>
  <title>Website Ayas</title>
  <body>
    <div>
      <h1>Halo, ini heading loh</h1>
      <p>Sebuah paragraf.</p>
    </div>
  </body>
</html>
```
